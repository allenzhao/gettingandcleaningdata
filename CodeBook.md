# Dataset Code book
## Summary

This dataset contains 10299 obeservations of 79 variables from 30 subjects with 6 activities. Other refs can be found in the original dataset's ([UCI HAR Dataset](https://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones)) readme.

## ID variables:

- `Subject` the ID of the experiements subject
- `Activity` the activity of the subject. Categorical Variable
	- WALKING : subject was walking during the test
	- WALKING_UPSTAIRS : subject was walking up a staircase during the test
	- WALKING_DOWNSTAIRS : subject was walking down a staircase during the test
	- SITTING : subject was sitting during the test
	- STANDING : subject was standing during the test
	- LAYING : subject was laying down during the test
